### Mọi người có tương tác với nhau
```kale
Nhi, Thao, Loc, Thinh, Tri, Phong, Quan, Nhan
Thao-Loc-Tri-Thinh-Thao-Tri-Nhi-Loc-Thinh
Phong-Quan-Tri-Phong-Thao
```
### Mọi người hiểu các ý tưởng, sản phẩm, nguồn lực của nhau
```kale
Nhi, Thao, Loc, Thinh, Tri, Phong, Quan, Nhan
Nhi-Loc-Tri-Thinh-Loc
Phong-Quan-Tri-Phong
```
### Mọi người hiểu thế giới quan của nhau
```kale
Nhi, Thao, Loc, Thinh, Tri, Phong, Quan, Nhan
Nhi-Loc-Tri-Thinh-Loc
Phong-Quan-Tri-Phong
```
